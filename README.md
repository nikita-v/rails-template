# Rails template

Template for a quick start.

## Template sructure:
* haml
* rspec
* guard
* factory_girl
* database_cleaner

## Usage

#### Run command for create project:

    $ rails new <project_name> -T -m https://raw.github.com/Nikita-V/rails-template/master/template.rb
